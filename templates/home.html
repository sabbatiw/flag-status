How should I display my flag?

This is an attempt to keep you up-to-date as to whether or not a flag should (officially) be flown at half- or full-mast. First, retrieve a list of supported flags, and then request the status of a flag. All responses are in valid json.

Example API Request/Response

curl https://lab.seekup.nl/flag-status/flags/us/
{
  "half_staff": false,
  "message": "success",
  "reason": "normal operations",
  "until": "indefinitely"
}

Report issues or request support for additional flags on github.
